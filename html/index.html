Калькулятор Галуа v1 by Artur Buriev

Как работает?

Данная программа полностью реализует функционал арифметики конечных полей (полей Галуа).
Принцип работы довольно простой:
1) выберите тип поля GF(p) или GF(p^m);
2) задайте необходимые параметры поля p и m (m задается при условии, что выбран тип поля GF(p^m));
3) нажмите кнопку "Сгенерировать поле";
4) будет сгенерировано конечное поле по заданным параметрам и построены матрицы операций;
5) чтобы получить результат операции для двух элементов, доступны два поля, в которые можно ввести элементы и выбрать операцию.

P.S.
Для поля GF(p) реализованы следующие операции: сложение, умножение, деление, вычитание, возведение в степень, взятие из-под корня.
Для поля GF(p^m): сложение, умножение, вычитание, деление.
